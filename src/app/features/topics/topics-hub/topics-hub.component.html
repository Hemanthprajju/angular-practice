Common interview questions covered

What is the difference between BehaviorSubject and Subject?
BehaviorSubject stores the latest value; new subscribers get it immediately. See AuthService.
Signals vs Observables?
Signals are synchronous, fine-grained, integrated with change detection. Observables excel at async streams.
CanActivate vs CanMatch?
CanActivate runs before route activation; CanMatch decides if a route config matches at all.
Standalone vs NgModule?
This app uses standalone components (Angular 15+) — no feature NgModules required.
